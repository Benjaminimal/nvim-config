# Title: Surround motions
# Category: Edit
# Tags: brackets, text-objects, add, delete, change
---
In normal mode use the following motions to operate on text objects surrounded by brackets, quotes, or other delimiters.

```vim
ys<TEXT_OBJECT><CHARACTER>  -- Add surrounding in Normal and Visual modes
ds<CHARACTER>               -- Delete surrounding
rs<CHARACTER><CHARACTER>    -- Replace surrounding
```
